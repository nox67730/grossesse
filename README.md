# 🌱 Suivi de grossesse — PWA

Application web progressive (PWA) de suivi de grossesse : bébé, nutrition personnalisée et courbe de poids maman.

## Fonctionnalités

- **Bébé** — illustration semaine par semaine + comparaison fruit rigolo animé
- **Courbe** — poids fœtal sur 40 semaines (données OMS Intergrowth-21)
- **Nutrition** — calories personnalisées selon IMC, poids et activité (formule Mifflin-St Jeor + OMS/ANSES/IOM)
- **Mon poids** — calendrier de saisie du poids maman + courbe de suivi personnelle

## Publier sur GitHub Pages

1. Créez un dépôt GitHub (ex: `suivi-grossesse`)
2. Uploadez tous les fichiers de ce dossier
3. Allez dans **Settings → Pages → Source → main**
4. Votre app sera disponible sur `https://votre-nom.github.io/suivi-grossesse`
5. Ouvrez l'URL dans Chrome Android → **"Ajouter à l'écran d'accueil"**

## Installation comme app Android

Une fois hébergée sur GitHub Pages, ouvrez l'URL dans **Chrome** sur Android.  
Chrome affiche automatiquement une bannière d'installation.

## Sources des données

- Poids fœtaux : **Intergrowth-21st / OMS** (Papageorghiou et al., BJOG 2014)
- Besoins caloriques : **ANSES / OMS**
- Prise de poids recommandée : **IOM (Institute of Medicine)**
- Métabolisme de base : **Formule Mifflin-St Jeor**
